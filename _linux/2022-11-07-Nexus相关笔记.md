## 安装

```sh
# 拉去镜像
$ docker pull sonatype/nexus3
# 创建数据卷目录
$ mkdir /some/dir/nexus-data && chown -R 200 /some/dir/nexus-data
# 运行docker镜像
$ docker run -d -p 9081:8081 --name nexus -v /some/dir/nexus-data:/nexus-data sonatype/nexus3
```
